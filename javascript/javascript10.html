.querySelector()

.querySelector()는 CSS 선택자로 요소를 선택하게 해줍니다. 주의할 점은 선택자에 해당하는 첫번째 요소만 선택한다는 것입니다.

.querySelectorAll()

.querySelectorAll()은 특정 CSS 선택자를 가진 모든 요소를 배열로 가져오는 메서드입니다.

.getElementById()

.getElementById()은 태그에 있는 id 속성을 사용하여 해당 태그에 접근하여 하고 싶은 작업을 할 때 쓰는 함수입니다 해당 id가 없는 경우 null 에러가 발생합니다.

.getElementClassName()

.getElementClassName()은 태그에 있는 class 속성을 사용하여 해당 태그에 접근하여 하고 싶은 작업을 할 때 쓰는 함수입니다 해당 class가 없는 경우 null 에러가 발생합니다.
